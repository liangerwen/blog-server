# 这是一个博客接口项目，为 [blog-client](https://gitee.com/liang_er_da/blog-client) 和 [blog-management](https://gitee.com/liang_er_da/blog-management) 提供接口

### 技术栈
* JavaScript
* Node.js
* [Egg](https://eggjs.org)
### 调试

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### 部署

```bash
$ npm start
$ npm stop
```

### 更多使用请点击 [Egg文档](https://eggjs.org)
